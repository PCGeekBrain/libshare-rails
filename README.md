# libshare-rails

## About

A smaple website for keeping track of various libraries languages and frameworks. Built on rails.

## Installation

1. clone the repo
2. `bundle install`
3. `rake db:migrate`
4. `rake db:seed`
5. `rails s`

### Note 

For github login support you will need GITHUB-KEY and GITHUB-SECRET set in your enviroment variables.
